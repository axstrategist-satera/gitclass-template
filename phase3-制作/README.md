# Phase 3：制作

実際に作って仕上げるフェーズです。**途中経過もどんどんコミット**してください。
「作りかけ → 直した → また直した」の履歴そのものが評価対象です。完成してから 1 回だけコミット、はもったいない出し方です。

## 提出物（共通＋種類別）

**共通（全員必須）**：`制作ふりかえり.md`（このフォルダに最初から入っているファイルに直接記入。ファイル名は変えない。最終締切までに）

**種類別（自分のものだけ）**：

### プログラミング
| コミットするもの | 形式 |
|---|---|
| ソースコード一式 | このフォルダの `src/` に入れる（用意済み）。`node_modules/` 等の自動生成物は入れない（.gitignore 済み） |
| 動かし方の説明 | `src/README.md`（用意済み）を起動手順・使い方に書きかえる |
| 実行画面のスクショ | PNG を 2〜3 枚 |

### 動画
| コミットするもの | 形式 |
|---|---|
| 完成版の絵コンテ（変更があれば更新） | PNG / JPG |
| 編集メモ | `編集メモ.md`（カットごとの編集内容・使った素材・BGM の出典） |
| サムネイル・代表フレーム | PNG を 3〜5 枚 |
| **動画本体** | **コミットしない**（.gitignore で除外済み）。学校の Google Drive にアップし、共有リンクを `制作ふりかえり.md` に貼る |

### デザイン
| コミットするもの | 形式 |
|---|---|
| 元データ | .psd / .ai / .fig / .xd など。**1 ファイル 50MB 以下なら**コミット可 |
| 書き出し画像 | PNG（提出用の最終書き出し。印刷物なら PDF も可） |
| 制作過程のスクショ | 途中段階を数枚（過程の証拠になる） |

### 3D モデリング
| コミットするもの | 形式 |
|---|---|
| モデルデータ | .blend / .scad（.blend1 などのバックアップは除外済み） |
| レンダー画像 | PNG を複数アングルで 2〜4 枚 |
| （3D プリントする場合） | .stl も追加 |

## ファイルサイズのルール（重要）

- **1 ファイル 50MB 以下**が目安。**100MB を超えるファイルは GitHub が受け付けず、コミットがエラーになります。**
- 50MB を超えそうなら、まず縮小を試す：
  - 画像 → 解像度を下げる／PNG を JPG にする
  - .blend → 不要オブジェクト削除、テクスチャを別ファイル化
  - .psd → レイヤー統合版を別名で書き出す
- どうしても 50MB 超のデータを履歴管理したい場合は **Git LFS** という仕組みがありますが、この授業では**使いません**（設定が難しく、無料枠の制限もあるため）。50MB 超は Google Drive リンクでの提出に切り替えて、先生に一言伝えてください。
