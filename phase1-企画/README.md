# Phase 1：企画

「何を・誰のために・なぜ作るか」を言葉にするフェーズです。
ここが固まっていないと Phase 2 以降で必ず迷子になります。じっくり考えて、何度も書き直して OK。

## 提出物

| ファイル | 内容 | 必須 |
|---|---|---|
| `企画書.md` | このフォルダに最初から入っているファイルに直接記入 | 必須 |
| 参考資料（画像・リンク集など） | 参考にした作品のスクショやメモ。`参考/` フォルダを作って入れる | 任意 |

## 手順

1. このフォルダの `企画書.md` を開いて、**そのまま直接**各項目を埋める（ファイル名は変えない。自動チェックがこの名前を見ています）
2. 最初は全部埋まらなくて OK。埋まったところまででまずコミット
3. コミットメッセージ例：`phase1: 企画書の初稿を作成`
4. フィードバックが返ってきたら、直してまたコミット：`phase1: フィードバックを受けて課題設定を具体化`

## このフェーズのポイント

- 「作りたいもの」だけでなく「**誰のどんな困りごと・楽しみのためか**」を書けているか
- 完成イメージが**他人に伝わる言葉**になっているか
- 実名・学校名を書いていないか（公開リポジトリです）
